# 🕹️ Space shooter & Catch the ball Game using Python Pygame 

A simple 2D Catch the ball game built with Python and Pygame.
The player controls a moving tower to catch falling blocks, increase score, and avoid missing too many objects before the game ends.

---

## 🚀 Project Overview

This project is a beginner-friendly game development project created using **Python** and **Pygame**. The game demonstrates core game programming concepts such as:

* Object-oriented programming
* Collision detection
* Sprite management
* Real-time game loops
* Event handling
* Score tracking

The objective is to catch falling blocks using the tower while preventing objects from falling off the screen.

---

## 🎮 Gameplay Features

* Smooth left/right tower movement
* Randomly generated falling blocks
* Real-time collision detection
* Score system
* Miss counter & game-over logic
* Dynamic object spawning
* FPS-controlled gameplay

# ▶️ How to Run

## 🔹 Step 1: Download the Project

Clone the repository or download it as a ZIP file and extract it.

---

## 🔹 Step 2: Open Terminal

Open a terminal or command prompt inside the project folder.

---

## 🔹 Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔹 Step 4: Run the Game

```bash
python "space shooter.py"
```

---

## 📦 Requirements

* Python 3.x
* Pygame 2.6.1

Install manually:

```bash
pip install pygame==2.6.1
```
